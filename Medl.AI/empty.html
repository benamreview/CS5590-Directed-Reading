Breast Cancer Detection

Model Description

Duy Ho

3/27/19

tensorflow python
  • Wiki
  • Publication
  • Prediction
  • Discussion

This model classifies the cardiomegaly from chest X-ray images. This model uses transfer learning by taking the first layers of a VGG16 model trained on ImageNet data and retrains on grayscale images of chests.

Dataset: Chest X-Rays

Accuracy: 82%

Dataset: https://www.kaggle.com/kmader/cardiomegaly-pretrained-vgg16/data

Source Code: https://www.kaggle.com/kmader/cardiomegaly-pretrained-vgg16/notebook
